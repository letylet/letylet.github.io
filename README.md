<h1>My Personal Portfolio 🚀</h1>

Welcome to the repository of my personal portfolio! This project is a reflection of who I am, blending clean code with interactive elements to create a unique digital experience.

<h2>🛠 The Tech Stack </h2>
To keep the site fast, responsive, and easy to maintain, I chose a modern web stack:

Framework: Next.js (React)
Styling: Tailwind CSS – for a lightweight, utility-first approach to design.


<h2>💡 The Workflow </h2>

I used Google AI Studio to generate the foundational structure of the site. I provided specific prompts regarding the aesthetic style, the detailed structure of each page, and the exact flow of user interactions I wanted to achieve.

Once the "bones" were ready, I took full control. I customized the code, refactored components, and added new features.
